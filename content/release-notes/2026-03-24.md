### **Crunch web application**

* Recent updates to the UI:
  * Updated the UI for User settings. See the [help article](https://help.crunch.io/hc/en-us/articles/360039103491-Changing-user-settings) for more information.
  * Updated the UI for Manage Users and added the ability for account admins to manage user email addresses. See the [help article](https://help.crunch.io/hc/en-us/articles/20214383713165-Managing-a-user-in-the-Manage-users-panel) for more information.
  * Made names and email addresses clickable links in folder permissions for account admins to navigate to the Manage Users page for the selected user.
  * Added a close button on the results panel for AI-Powered Search.
* Changed the default search from legacy search to the new AI-Powered Search for all users in Crunch.
* Fixed a bug that corrects user permissions for users who have Edit permission on a folder without Manage Users permission.

### **Crunch Automation**

* Added the ability to alter an email address to the [ALTER USERS command](https://help.crunch.io/hc/en-us/articles/21986841356301-ALTER-USERS-command).

### **Lakehouse**

* Added the ability to compare audiences through named expressions to the [Lakehouse Tabulation API](https://help.crunch.io/hc/en-us/articles/43012931204877-Tabulating-Data-on-the-Lakehouse#toc4).
* Implemented significant performance improvements on the [Lakehouse Download API](https://help.crunch.io/hc/en-us/articles/40974722113293-Exporting-Data-with-Crunch-Lakehouse-APIs#toc0).
