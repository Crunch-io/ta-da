### **Crunch**

* Added functionality to highlight search matches for variables and documents, including synonyms.
* Updated the folder sidebar and standardized drag and drop. Users can now drag and drop content inside the folder sidebar and inside individual folders.
* Folder properties have been updated for a more consistent and reliable experience.
* Introduced the AI Agent for all datasets across Crunch. The AI Agent lets users ask questions in plain language and get instant, accurate answers from data in Crunch. See the [AI Agent](https://help.crunch.io/hc/en-us/articles/44646135976461-Crunch-AI-Agent) helpdesk article to learn more.
* Released the new Explore Mode experience for all Crunch users, replacing Variable Summaries, designed to help users discover insights and navigate even the most complex datasets seamlessly. See the [Browsing Data in Explore Mode](https://help.crunch.io/hc/en-us/articles/43511673039117-Browsing-Data-in-Explore-Mode) helpdesk article to learn more.

### **Lakehouse**

* New optimized Datasource Append API. [Datasource Append API Reference](https://crunch.io/api/reference/#post-/datasources/-datasource_id-/append_data/).
* Datasource Tabulation API now supports Date/Time variable types.
* Improved performance on Lakehouse SPSS Importer.
