### **Crunch web application**

- The following features are now available in *wide access*:
    - The new variable sidebar
    - The filter bar refresh
- Added an **Explore** tab, which [displays a summary](https://crunch.io/dev/features/new-way-to-explore-data/) of every variable in your dataset — the individual tiles now provide an instant top-line visualization of the data (in *early access*). See the [help article](https://help.crunch.io/hc/en-us/articles/43511673039117-Browsing-Data-in-Explore-Mode) for more information.
- Unified navigation header and refined dashboard navigation for improved consistency.
- Added support to include subvariables in search results by folder with the AI-Powered Variable Search. See the [help article](https://help.crunch.io/hc/en-us/articles/33552869343117#avail-subvar) for more information.

### **API endpoints and API reference**

Deprecated the following API endpoints: 

- [List users with dataset access](https://crunch.io/api/reference/#get-/datasets/-dataset_id-/users/)
- [List all users with dashboard access](https://crunch.io/api/reference/#get-/dashboards/-dashboard_id-/users/)

### **Lakehouse**

Added a [Tabulation API](https://help.crunch.io/hc/en-us/articles/43108717800461-Tabulation-API) which is a flexible export interface for aggregated survey data, providing:

- **Asynchronous batch cross-tabulations** with any combination of row and column variables.
- **Row-level filtering** using a declarative syntax.
- **Long-form output** designed for ingestion into reporting pipelines.

### **Backend/CrunchDB**

- Updated the `copy_from` command so that it no longer supports the ability to copy dataset shares.
